TECH STACK OF FULL STACK WEB DEVELOPMENT

JavaScript
A programming language that conforms to the ECMAScript specification.
HTML
The standard markup language for documents designed to be displayed in a web browser.
AJAX
Asynchronous JavaScript and XML enable making requests to the server without refreshing the entire page.
RESTFUL API
An application program interface (API) that allows applications to request and send data from a server over HTTP using either XML or JSON formats
WEB ASSEMBLY
A binary instruction format that enables running performance language like C and C++ in the browser
